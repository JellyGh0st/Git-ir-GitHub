Git

Git'as yra populiariausia kodo versijavimo programa.

Git Komandos

  • init - sukuria git'o repozitoriją
  • add - prideda nurodytus failus prie commit'ui paruoštų failų (staged)
  • status - nurodo informaciją apie tavo lokalios git'o repozitorijos statusą
  • commit -m "žinutė" - sukuria kodo versiją iš stage'intų failų ir prideda žinutę apie pakitimus (žinutę bandyti išlaikyti ne ilgesnę negu 50 simbolių)
  • push - iš lokalios repozitorijos commit'ų nusiuntimas į išorinę repozitoriją
  • pull - iš išorinės repozitorijos commit'ų atsisiuntimas į lokalią repozitoriją
  • remote add origin https://... - išorinės repozitorijos prijungimas prie lokalios repozitorijos
  • log - commit'ų sąrašas
  • checkout branchName|commitID - Pereiti į kitą kodo šaką arba versiją
  • branch naujosSakosPavadinimas - Sukuria naują šaką
  • clone https://... - Nuklonuoja išorinę repozitoriją

GitHub
